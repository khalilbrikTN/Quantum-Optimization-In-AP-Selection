# Quick Start Guide

## For Jupyter Notebooks (Most Common)

1. **Start Jupyter**:
   ```bash
   cd "c:\Users\Mohamed Khalil\Desktop\Quantum-Optimization-In-AP-Selection"
   venv\Scripts\activate
   jupyter notebook
   ```

2. **In Jupyter**:
   - Open your notebook
   - Click kernel selector (top right)
   - Select: **"Python (Quantum VEnv)"**
   - Run your cells!
